About

[2017.08.12]
一直想寫一個技術相關的blog，不過總是給自己找了好多理由沒去做。

收集了不少工作上的筆記，卻很少去 review 再加上自己記憶力也不太好，
以至於每次別人問我，你以前都幹了什麼好事，解了什麼 bug，
我的腦子裡都是一片空白。

這個 blog 從建立到現在荒廢放了很久，
以前也用過無名小站、google blog、FB 之類的來寫一些東西，
不過都不是很有系統的整理......

架一個網站，有自己的 domain，
感覺就像工程師會做的事情，
而且可以建立一下個人品牌XD

之前在一些免費的 PAAS 服務上，像是 openshift，也架過 wordpress
不過一樣沒有持續下去，而且免費服務其實也不太穩定。

看了下 github 的紀錄，2016 的 12 月，我就已經建好部落格。
這八個月來，發生了一些事，我的 blog 毫無進展，世界停止轉動。
今天開始要忘記那些不開心的事，『用你的微笑改變世界，不要讓世界改變了你的微笑。』

Fallrising
